## المحتوى

### القرار A-001

العنوان:

Child Journey Core هو القلب المعماري لمنصة عونك.

الحالة:

Approved

التاريخ:

2026-07-06

القرار

تم اعتماد Child Journey Core كنواة معمارية رئيسية للنظام.

جميع الأنظمة الأخرى تُبنى لخدمة رحلة الطفل وليس العكس.

السبب

تنظر عونك إلى الطفل باعتباره رحلة نمو مستمرة وليست ملفًا ثابتًا.

التقييم، والخطة، والجلسات، والتقارير، والذكاء الاصطناعي، وجميع الأنظمة الأخرى هي أحداث أو أدوات ضمن هذه الرحلة.

التأثير المعماري

هذا القرار يؤثر مستقبلاً على:

- System Architecture
- Database Design
- Data Flow
- AI Engine
- Reporting
- Assessment
- Rehabilitation
- Family Module

البدائل التي تمت مناقشتها

- Child Core System
- Child Record
- Child Profile

وتم رفضها لأنها تنظر إلى الطفل كسجل ثابت وليس كرحلة مستمرة.

النتيجة

اعتبار Child Journey Core المرجع الأساسي الذي تدور حوله جميع الأنظمة الأخرى.
